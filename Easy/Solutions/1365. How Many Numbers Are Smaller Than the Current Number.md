## Java

```Java
class Solution {
    public int[] smallerNumbersThanCurrent(int[] nums) {
        int [] arr=new int[nums.length];
        for(int i =0;i<nums.length;i++){
            int temp = 0;
            for(int num:nums){
                if(nums[i]>num){
                    temp++;
                }
            }
            arr[i] = temp;
        }
        return arr;
    }
}
```
