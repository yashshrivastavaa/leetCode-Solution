## Java

```Java
class Solution {
    public int specialArray(int[] nums) {
        int[] arr = new int[1001];
        for(int num : nums)
            arr[num]++;
        int total = nums.length;
        for(int i = 0;i<1001;i++) {
            if(total == i)
                return i;
            total-=arr[i];
        }
        return -1;
    }
}
```
