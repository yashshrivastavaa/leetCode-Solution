## Java

```Java
class Solution {
    public int findNumbers(int[] nums) {
        int digit=0;
        for(int i =0;i<nums.length;i++){
            if((nums[i]<=999999 && nums[i]>=100000) || (nums[i]<=9999 && nums[i]>=1000) || (nums[i]<=99 && nums[i]>=10)){
                digit++;
            }
        }
        return digit;
    }
}
```
