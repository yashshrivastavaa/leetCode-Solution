## Java

```Java
class Solution {
    public boolean checkIfExist(int[] arr) {
        for(int i = 0;i< arr.length-1;i++){
            for(int j = i + 1;j<arr.length;j++) {
                if( (float)arr[i] / 2 == (float)arr[j] || arr[i] * 2 == arr[j]) {
                    return true;
                }
            }
        }
        return false;
    }
}
```
